# Apriori-algorithm
Enter the minimum support, confidence and fift, for example: 0.9, 0.8, 1. 

### The program outputs association rules for the types of defects in the lamination process.

 - A - air bubles
 - B - burnt surfaces
 - C - surpluses to grind
 - D - wrinkling
 - E - scratches 
 - F - cracks
 - G - inprint
 - H - deformation 
 - I - demould defects
 - J - delaminations
 - K - porosity
 - L - dimplings
 - M - dry reinforcement
 - N - spots
 - O - anti-sliding defects
 - P - matte finish
 - Q - other defects
